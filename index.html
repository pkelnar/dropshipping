Jeden link, oba vidíte stejný stav. Potřebuješ Firebase projekt — trvá 3 minuty a je zdarma.
Krok 1 — Jdi na console.firebase.google.com, vytvoř nový projekt (libovolné jméno).
Krok 2 — V projektu klikni na Realtime Database → Create database → zvol Start in test mode.
Krok 3 — Zkopíruj URL databáze. Vypadá takto:
https://tvoj-projekt-default-rtdb.firebaseio.com
Krok 4 — Vlož URL sem a zadej svoje jméno. Pak pošli kamarádovi stejný soubor — ten zadá tu samou URL a svoje jméno.
